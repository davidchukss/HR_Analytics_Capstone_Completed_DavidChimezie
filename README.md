# HR Analytics Dashboard (Excel)
## Project Overview
This project is an interactive HR Analytics Dashboard built entirely in Microsoft Excel to analyze workforce data and provide insights for HR decision - making

The dashboard transforms raw employee records into meaningful visualizations using Pivot Tables, Pivot Charts, Slicers, formulas, and KPI cards.

This project was completed as part of my Data Analytics class at IOTB Tech to demonstrate my Excel dashboarding and business analysis skills.

## Project Obejectives
The dashboard answers key HR business questions such as:
- What is the total employee headcount?
- What is the total salary expenditure?
- What is the average employee salary?
- Which departments have the highest number of employees?
- How are employees performing across the organization?
- How many employees are Active, Terminated, or On Leave?

## Dashboard Features
### The dashboard includes:
#### Interactive KPI Cards
- Total Employees
- Total Salary
- Average Salary
- Total Departments

#### Interactive Charts
-Employee Distribution by Department
- Salary by Department
- Performance Rating Distribution
- Employment Status Breakdown
- Hiring Trend by Year
  
#### Dynamic Filters (Slicers)
- Department
- Employment Status

Users can filter the dashboard to instantly update all visuals and KPIs.
